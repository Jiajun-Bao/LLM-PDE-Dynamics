# Text-Trained LLMs Can Zero-Shot Extrapolate PDE Dynamics
This repository contains the code to reproduce all data and figures from “Text-Trained LLMs Can Zero-Shot Extrapolate PDE Dynamics” by Jiajun Bao, Nicolas Boullé, Toni J.B. Liu, Raphaël Sarfati, and Christopher J. Earls. The code will be released publicly upon publication of this work.
